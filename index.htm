Tags

(Voltar)

As tags são utilizadas para informar ao navegador a estrutura do site, ou seja, as Tags serão interpretadas pelo navegador, produzindo a estrutura e o conteúdo visual do site. Um documento HTML recebe algumas tags que formam a sua estrutura básica. As tags estão sempre dentro dos sinais de "maior que" e "menor que", ou seja: < >.

As tags de título possuem valor semântico, variando entre 6 níveis hierárquicos. Para definir títulos utilizamos as tags que vão de "h1" a "h6". A distribuição de título interfere na otimização de uma página. Uma página contendo apenas "h1" não seria bem vista pelos mecanismos de busca. Portanto, é importante dividir os títulos deixando o "h1" apenas para o título principal, "h2" ou "h3"para título das seções e "h4" a "h6" para subtitulos ou títulos que possuam baixo valor hierárquico em relação aos demais.

As tags de texto definem textos, estilos da fonte, parágrafos, spans, quebra de linhas e entre outros. Algumas dessas tags são:

- p : Principal tag de texto, compõe um parágrafo;

- u : Conteúdo exibido com um sublinhado;

- i : Transforma o conteúdo em itálico;

- strong : Transforma o conteúdo em negrito;

- sup : Define o texto sobrescrito, ele aparece meio caractere acima da linha normal;

- blockquote : Citação de outro autor no meio do texto;

- small : Define um texto menor;

- abbr : Define uma abreviação ou um acrônimo como "HTML", "CSS", "Mr";

- sub : Define o texto sobrescrito, ele aparece meio caractere abaixo da linha normal;

- mark : Define o texto que deve ser marcado ou destacado;

- font : foi usada no HTML 4 para especificar a face da fonte, o tamanho e a cor do texto;

- del : Define o texto que foi excluído de um documento.

- hr/ : Não necessita de fechamento, ela forma uma linha horizontal.

Atributos das Tags

(Voltar)

Os atributos são utilizados para deixar as tags personalizadas, modificando sua estrutura ou funcionalidade. Alguns deles são:

- class : Atribui uma classe ao elemento;

- id : Atribui um id ao elemento, um id deve ser atribuído a um único elemento, deve ser único;

- style : Permite incluir comandos de CSS;

- title : Define o título do elemento;

- widht : Define uma largura para o elemento;

- height : Define uma altura para o elemento;

Listas Ordenadas e não Ordenadas

(Voltar)

Para criar uma lista, utiliza-se a tag "ol" para uma lista ordenada, ou a tag "ul" para lista não ordenada. Posteriormente, inclui-se dentro da lista os elementos da mesma, dentro das tags "li".

Link

(Voltar)

A tag de link HTML é responsável que faz a ligação entre um documento e outro sendo ele da mesma página ou de uma página de outro domínio. Para realizar um link, podemos chamar as tags "a" com o atributo href.

Para criar um link no texto que redirecione à página inicial da DIO por exemplo, ficaria assim:
